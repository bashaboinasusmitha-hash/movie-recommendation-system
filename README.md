# Movie Recommendation System

##  Description
This project recommends movies based on genre similarity using cosine similarity.
## Technologies Used
- Python
- Pandas
- Scikit-learn
## Features
- Recommends similar movies based on genre
- Uses cosine similarity algorithm
- Simple and efficient implementation
## How to Run
1. Install required libraries:
   pip install pandas scikit-learn
2. Run the Python file:
   python movie_recommendation.py
3. Enter a movie name to get recommendations
## Output
Displays top recommended movies based on similarity

## Dataset
- Contains movie titles and genres
